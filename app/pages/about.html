Poke Pathfinder

Poke Pathfinder is a short demo inspired by the widely popular franchise, Pokemon. This demonstration allows users to use pathfinding algorithms to explore an infamous location in Pokemon world, Cerulean Cave.

This project is an excellent tool for teaching others about the mechanics of pathfinding algorithms. There are many features that allow users to tweak which alorithm is used and how it is implement.

Using Poke-Pathfinder is easy and fun!

    1. Select the algorithm

    There are four algorithms to choose from: Breadth-first search, Depth-first Search, Dijksta's and A*
